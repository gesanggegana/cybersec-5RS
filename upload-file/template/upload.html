Upload File (Vulnerable Demo)

Catatan keamanan: Form ini tidak melakukan validasi nama/tipe file dan menyimpan file dengan nama asli. Hanya untuk latihan di lingkungan terisolasi.


Petunjuk latihan: coba upload file bernama ../../etc/passwd atau upload webshell (hanya dalam VM/isolated env) untuk melihat bagaimana path traversal / unsafe upload bisa dieksploitasi.

Untuk mitigasi singkat: batasi ekstensi/MIME, simpan dengan nama acak (UUID), dan simpan di direktori non-eksekusi.
